# Hello Astro

Astro is a modern web framework designed for **content-driven websites**. It ships zero JavaScript by default, making your pages lightning fast.

## Why Astro?

- Static HTML output
- Bring your own UI framework (React, Vue, Svelte...)
- Built-in Markdown support
- Islands architecture for selective hydration
