# A Photo Gallery Built with Islands Architecture

This post demonstrates how Astro's **Islands Architecture** works in practice. Everything you're reading right now — the layout, the text, the styles — is **pure static HTML**. Zero JavaScript was loaded to render it.

The gallery below is the island: a React component that ships its own JavaScript bundle and hydrates independently, without affecting the rest of the page.

## How it works

With MDX, the component is imported and placed directly in the post body — no frontmatter tricks, no logic in `[slug].astro`:

```mdx
import ImageGallery from "../../components/ImageGallery";

<ImageGallery client:visible images={[...]} />
```

The `client:visible` directive tells Astro to defer hydration until the gallery scrolls into the viewport.
